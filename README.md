# Fuzzy Logic HVAC Control
An HVAC control based on Fuzzy Logic Implementation.

## Setup
Pre-requisite: Python (Python 3.12.9) was installed and configured in the local machine.

after cloning the code into local directory, locate into the dir: cd fl-hvac-control
create virtual environment (to abstract pip installation): python -m venv venv
activate the virtual environment (bash cmd): source venv/bin/activate
activate the virtual environment (windows): venv\Scripts\activate
install required libraries: pip install -r requirements.txt
